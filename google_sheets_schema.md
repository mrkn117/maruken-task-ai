# Googleシート構造定義

スプレッドシート名：電気工事_空き時間タスク管理

---

## 1. employees シート

社員マスタ。手動で登録・更新する。

| 列名 | 型 | 必須 | 説明 |
|---|---|---|---|
| employee_id | 文字列 | ✅ | 社員ID（例: E001）。重複不可 |
| 社員名 | 文字列 | ✅ | フルネーム（例: 田中 太郎） |
| レベル | 数値 | ✅ | 1〜7（employee_levels.md参照） |
| 得意分野 | 文字列 | - | 得意な作業カテゴリ（カンマ区切り可） |
| 苦手分野 | 文字列 | - | 苦手な作業カテゴリ（カンマ区切り可） |
| 性格傾向 | 文字列 | - | 性格・行動特性（カンマ区切り可） |
| 現在の状態 | 文字列 | ✅ | 在籍中 / 休職中 / 退職済み |
| 最終更新日 | 日付 | ✅ | YYYY-MM-DD形式 |

---

## 2. task_history シート

社員ごとのタスク実施履歴。AIと社員・上司が記録する。

| 列名 | 型 | 説明 |
|---|---|---|
| task_id | 文字列 | タスクID（例: T1746000000000）自動生成 |
| employee_id | 文字列 | 社員ID（employeesと紐付け） |
| 社員名 | 文字列 | 社員名（読みやすさのため） |
| 実施日 | 文字列 | 実施日（YYYY/M/D形式） |
| 空き時間 | 文字列 | 15分 / 30分 / 1時間 など |
| 場所 | 文字列 | 事務所 / 倉庫 / 資材置き場 / 社内 |
| タスク名 | 文字列 | AIが生成したタスク名 |
| カテゴリ | 文字列 | タスクカテゴリ（task_rules.md参照） |
| AI指示内容 | 文字列 | AI出力全文（先頭500文字） |
| 成果物 | 文字列 | 社員が記入した成果物メモ |
| 状態 | 文字列 | 未実施 / 途中 / 不十分 / 完了 |
| 評価 | 文字列 | 1点〜5点 |
| 上司コメント | 文字列 | 上司からのコメント |
| 次回への申し送り | 文字列 | 次回指示に引き継ぐ事項 |

---

## 3. ai_task_output シート

AIが出力したタスク指示の詳細ログ。自動書き込み。

| 列名 | 型 | 説明 |
|---|---|---|
| created_at | 日時 | 生成日時（ISO 8601形式） |
| employee_id | 文字列 | 社員ID |
| 社員名 | 文字列 | 社員名 |
| 判定レベル | 数値 | AIが参照した社員レベル |
| 空き時間 | 文字列 | 入力された空き時間 |
| 場所 | 文字列 | 入力された現在地 |
| 判断結果 | 文字列 | 新規タスク / 再指示 / レベルアップタスク |
| 優先課題1 | 文字列 | AIが判断した優先課題1位 |
| 優先課題2 | 文字列 | AIが判断した優先課題2位 |
| 優先課題3 | 文字列 | AIが判断した優先課題3位 |
| 今すぐやるタスク | 文字列 | タスク名 |
| 成果物 | 文字列 | 指定した成果物 |
| 完了条件 | 文字列 | 完了の判定条件 |
| 次タスク | 文字列 | 完了後の次タスク |
| 未完了時の再指示 | 文字列 | 未完了・不十分時の再指示内容 |
| 完了時の次ステップ | 文字列 | 完了時の次ステップ |

---

## 4. task_categories シート

タスクカテゴリのマスタ。参照用。

| 列名 | 説明 |
|---|---|
| カテゴリ名 | 工具管理 / 材料管理 / 安全管理 など |
| 対象レベル | Lv1〜Lv7 |
| 目的 | このカテゴリの目的 |
| 会社メリット | 会社にとっての利益・効果 |
| 例タスク | 具体的なタスク例 |

---

## 5. evaluation_rules シート

評価基準のマスタ。参照用。

| 列名 | 説明 |
|---|---|
| 評価項目 | 評価する観点 |
| 点数 | 1〜5 |
| 条件 | その点数の基準 |
| 備考 | 追記事項 |
